# King County House Sales Analysis

## Project Overview

This project analyzes house sales data from **King County, USA**. The analysis focuses on understanding housing price patterns and the factors that influence property prices.

The project was completed using **Tableau** to create interactive dashboards that help visualize housing market trends.

---

## Business Problem

Understanding housing price trends is important for real estate companies, investors, and potential home buyers.

This project explores several questions:

* How are house prices distributed in King County?
* Which areas have higher property prices?
* What house characteristics may influence the price?
* How do housing features such as bedrooms, bathrooms, and living area relate to house prices?

---

## Dataset

The dataset used in this project is **King County House Sales Data**.

The dataset contains information about house sales in King County, including:

* House price
* Number of bedrooms
* Number of bathrooms
* Living area size
* Lot size
* House condition
* Year built
* Location (zipcode, latitude, longitude)

Total records: **21,060 rows**

---

## Tools Used

The analysis in this project was conducted using:

* **Microsoft Excel** (dataset)
* **Tableau** (data visualization and dashboard)

---

## Project Workflow

### 1. Data Understanding

The dataset was reviewed to understand the available variables and general structure of the data.

### 2. Data Exploration

Exploration was conducted to observe the distribution of house prices and house characteristics.

### 3. Data Visualization

Tableau was used to build visualizations that help reveal patterns in the housing market.

The dashboard includes several visualizations to explore:

* Price distribution
* House characteristics
* Location-based analysis
* Relationships between house features and price

---

## Key Insights

Some insights observed from the analysis include:

* House prices vary significantly across different locations in King County.
* Houses with larger living areas tend to have higher prices.
* The number of bathrooms and bedrooms also shows a relationship with house prices.
* Certain zip code areas appear to have higher average house prices.
